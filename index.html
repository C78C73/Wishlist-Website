One-time Firebase Setup

Wishlist needs a free Firebase project to store and share data between devices. Takes about 5 minutes.

Step 1 — Create a Firebase project

Go to console.firebase.google.com → Add project → Enter a name → Continue (you can disable Google Analytics)

Step 2 — Add a Web App

In your project dashboard: Click the </> (Web) icon → Give it a nickname → Register app → Copy the firebaseConfig object

Step 3 — Enable Firestore

In the left sidebar: Build → Firestore Database → Create database → Start in test mode → Choose a region → Enable

Step 4 — Paste your config here

The config looks like:

{ "apiKey": "AIzaSy...", "authDomain": "your-app.firebaseapp.com", "projectId": "your-project-id", "storageBucket": "your-app.appspot.com", "messagingSenderId": "123456789", "appId": "1:123456:web:abc123" }

Your config is stored in your browser only and is never sent anywhere else.
